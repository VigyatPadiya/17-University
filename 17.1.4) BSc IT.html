BSc IT Subjects

1. Programming Fundamentals

Introduction to programming concepts, algorithms, and problem-solving techniques using languages like Java or Python.

2. Web Development

Creating dynamic and interactive websites using HTML, CSS, JavaScript, and web frameworks.

3. Database Management

Designing, implementing, and managing databases using SQL and learning about database systems.

4. Data Structures and Algorithms

Study of fundamental data structures and algorithms for efficient information processing.

5. Networking Basics

Understanding computer networks, protocols, and network administration.

6. Software Engineering

Principles of software development life cycle, project management, and software design patterns.

7. Operating Systems

Exploring the functions and components of operating systems, including memory management and file systems.

8. Cybersecurity

Introduction to cybersecurity principles, threats, and measures to protect digital assets.

9. Mobile App Development

Creating applications for mobile devices using platforms like Android or iOS.

10. Cloud Computing

Understanding cloud technologies, virtualization, and cloud-based services.

11. Artificial Intelligence

Exploring AI concepts, machine learning, and applications of AI in various fields.

12. Project Management

Learning project management methodologies and practices for IT projects.
