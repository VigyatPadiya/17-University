BSc IT Data Science

BSc IT Data Science is an interdisciplinary program that combines information technology and data science concepts. It focuses on harnessing the power of data to gain insights and make informed decisions.

Key Subjects

  • Introduction to Data Science

    An overview of data science, its applications, and the data lifecycle. Introduction to tools and techniques for data analysis and visualization.

  • Programming for Data Science

    Foundational programming skills in languages like Python and R. Writing code to manipulate and analyze data.

  • Database Management

    Design and management of databases to store, retrieve, and organize large datasets. SQL and NoSQL databases.

  • Machine Learning

    Study of algorithms and models for machine learning. Supervised and unsupervised learning, feature engineering, and model evaluation.

  • Data Visualization

    Creating visual representations of data to communicate insights effectively. Use of tools like Matplotlib, Seaborn, and Tableau.

  • Big Data Technologies

    Exploration of big data frameworks like Hadoop and Spark. Processing and analyzing massive datasets.

Graduates of the BSc IT Data Science program are equipped with skills to work as data analysts, data engineers, business analysts, and more, across various industries.
